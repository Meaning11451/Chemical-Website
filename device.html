PCT-100 化工控制实训装置

基于 ESP32-03 · WiFi / MQTTs 通信 · 本地按键控制 · 智能报警系统

📋 产品概览

装置简介

PCT-100 是一款集温度检测、光照检测、OLED 显示、智能报警 于一体的化工控制实训装置。装置采用 ESP32-03 WiFi 模块 作为主控芯片，支持 本地按键 与 MQTTs 远程控制 两种操作方式，适用于化工自动化、嵌入式系统等课程的实训教学。

📐 技术规格
产品型号PCT-100
主控芯片ESP32-03（双核 Xtensa LX6，240 MHz）
WiFi 模块集成 2.4 GHz WiFi（802.11 b/g/n）
通信协议MQTTs（TLS 加密） / 本地按键
传感器数字温度传感器 · 光照强度传感器
显示OLED 屏幕（128×64）
执行器散热风扇 · LED 照明灯 · 总闸开关
报警方式蜂鸣器 + LED 闪烁
工作电压DC 5V / USB Type-C 供电
工作温度-10°C ~ 60°C
